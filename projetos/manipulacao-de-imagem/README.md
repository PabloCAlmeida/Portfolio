# 🎨 Manipulação de Imagem

## 📖 Sobre o projeto

Este projeto foi desenvolvido como atividade acadêmica da disciplina de Manipulação de Imagem, com o objetivo de aplicar técnicas de edição fotográfica utilizando o Adobe Photoshop.

Durante o desenvolvimento foram utilizadas ferramentas de seleção, ajustes de cores, brilho, contraste e organização em camadas para aprimorar a qualidade visual da imagem.

---

## 🎯 Objetivos

- Aplicar técnicas de tratamento de imagens.
- Aprender o fluxo de trabalho no Adobe Photoshop.
- Desenvolver atenção aos detalhes durante a edição.
- Produzir uma composição visual com acabamento profissional.

---

## 🛠 Ferramentas utilizadas

- Adobe Photoshop
- Camadas (Layers)
- Máscaras
- Ferramentas de Seleção
- Ajustes de Cor
- Brilho e Contraste

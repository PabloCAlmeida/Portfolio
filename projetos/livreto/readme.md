# Livreto

## Descrição
Projeto de diagramação e design editorial para criação de um livreto, explorando organização de conteúdo, composição de páginas e identidade visual.

## Objetivo
Desenvolver uma publicação impressa aplicando conceitos de design editorial, incluindo estrutura de páginas, tipografia, imagens e elementos gráficos.

## Ferramentas utilizadas
- Adobe InDesign
- Adobe Illustrator
- Adobe Photoshop

## Processo de criação

### Planejamento
Definição da estrutura do livreto, quantidade de páginas e organização das informações.

### Diagramação
Criação do grid, escolha tipográfica, distribuição dos textos e posicionamento dos elementos visuais.

### Finalização
Ajustes finais, revisão visual e preparação para impressão.

## Aprendizados
- Design editorial
- Diagramação de páginas
- Organização de conteúdo
- Preparação de arquivos para impressão

# Pôster

## Descrição
Projeto de criação de pôster desenvolvido para explorar composição gráfica, hierarquia visual e comunicação através de elementos visuais.

## Objetivo
Criar uma peça gráfica atrativa, utilizando conceitos de design como tipografia, equilíbrio, contraste e organização dos elementos.

## Ferramentas utilizadas
- Adobe Illustrator
- Adobe Photoshop (se utilizado)

## Processo de criação

### Conceito
Definição da ideia principal do pôster, estilo visual e elementos que seriam utilizados.

### Desenvolvimento
Criação da composição, escolha das cores, aplicação da tipografia e organização dos elementos gráficos.

## Aprendizados
- Composição visual
- Criação de peças de comunicação
- Uso de tipografia e elementos gráficos
- Desenvolvimento de uma identidade visual para uma arte única

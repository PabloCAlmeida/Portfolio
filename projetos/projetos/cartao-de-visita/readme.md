
# 💼 Cartão de Visita

## 📖 Sobre o projeto

Este projeto consistiu na criação de um cartão de visita com foco em identidade visual, organização das informações e comunicação profissional.

O desenvolvimento envolveu a aplicação de conceitos de composição gráfica, hierarquia visual, tipografia e equilíbrio entre os elementos, resultando em um material de apresentação funcional e visualmente agradável.

---

## 🎯 Objetivos

- Desenvolver um material gráfico para impressão.
- Aplicar princípios de identidade visual.
- Organizar informações de forma clara e objetiva.
- Trabalhar com tipografia e composição.

---

## 🛠 Ferramentas utilizadas

- Adobe Illustrator
- Tipografia
- Vetorização
- Alinhamento e Grid
- Composição Gráfica
